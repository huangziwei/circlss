# Package index

## Response families

mgcv general families for angular responses: one linear predictor — and
so one formula, with smooths — per distribution parameter.

- [`vmlss()`](https://huangziwei.github.io/circlss/reference/vmlss.md) :
  von Mises location-scale family for mgcv GAMs
- [`pnlss()`](https://huangziwei.github.io/circlss/reference/pnlss.md) :
  Projected normal location family for mgcv GAMs
